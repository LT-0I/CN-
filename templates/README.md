# 模板文件放置说明

请将你的导学案 Word 模板文件（.docx）放在此目录下。

## 要求

1. 模板文件必须是 `.docx` 格式（不支持 `.doc`）
2. 模板中应包含至少一个完整课时的导学案示例
3. 模板中的表格结构应保持完整（不要删除空行或空列）

## 使用方式

生成脚本会解包模板文件，提取以下格式参数：
- 页面尺寸和页边距
- 表格列宽和行高
- 字体、字号
- 段落缩进和行距
- 页眉内容

然后按照提取的参数生成新的导学案文档。

## 注意

- 模板文件本身不会被修改
- `.gitignore` 已配置忽略 `.docx` 文件，请手动管理模板文件
